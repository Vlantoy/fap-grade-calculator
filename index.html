Tính điểm thành phần & GPA kỳ trên FAP

Userscript cho trình duyệt: gõ thử điểm vào cột Value trên StudentGrade.aspx, tự tính Average theo trọng số, xử lý Resit, và gom TB các môn trong session thành GPA. Không gửi dữ liệu ra ngoài — chỉ chạy trên máy bạn.

Bắt đầu cài đặt Console MVP (1 môn) Xem source script
✏️

Nhập Value trên bảng FAP

Ô trống thành input. Điểm đã có trên FAP được giữ. Reload trang chi tiết có thể khôi phục từ session tab.

⚖️

Weight + Resit

Average = Σ(value × weight / 100). Có điểm thi lại → bỏ lần 1 cùng loại (Practical / Theory…).

📊

GPA kỳ (session)

Lưu TB từng môn trong tab → trang tổng quát hiện TB hệ 10 và GPA thang 4.0. Đóng tab = mất.

Cài đặt Tampermonkey

Bản đầy đủ: 1 môn + resit + GPA kỳ.

  1. Cài Tampermonkey (Chrome / Edge / Firefox).
  2. Tắt script kiểu “FAP Grade Watcher” (overlay đỏ + chuông) nếu đang bật — dễ đụng độ.
  3. Bấm Cài script (hoặc raw GitHub) → Tampermonkey hỏi cài → Install.
  4. Mở FAP Mark Report → chọn môn → gõ Value.
⚠️ Chỉ dùng để ước lượng / what-if. Điểm chính thức vẫn do nhà trường ghi trên FAP. Không share cookie / session FAP.
